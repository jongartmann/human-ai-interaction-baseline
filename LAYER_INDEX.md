# Baselines Suite (v0.1)

This repository contains **artifact-only baselines** for rolling out AI in real work — without hype and without product claims.

✅ Interface-neutral  
✅ Copy/paste templates  
✅ Anti-pattern galleries  
✅ Operator cards  
✅ Rollout checklists  

---

## Available layers

### 1) Human–AI Interaction Baseline (v0.1)
Defines how AI should behave in real collaboration:
- when it should **explain**
- when it should **defer**
- when it must **escalate**
- when it should be **quiet**

Start here: `layers/human-ai-interaction-baseline/v0.1/BASELINE.md`  
Anti-patterns: `layers/human-ai-interaction-baseline/v0.1/anti-patterns/ANTI_PATTERNS.md`  
Operator cards: `layers/human-ai-interaction-baseline/v0.1/operator-cards/README.md`  

---

### 2) Security & Stability Baseline (v0.1)
Prevents silent failure, drift, and invisible responsibility shifts:
- explicit **decision boundaries** (B0–B3)
- predictable **escalation paths**
- **verification hooks** for high-impact actions
- calm, design-only **kill switch** model

Start here: `layers/security-stability-baseline/v0.1/BASELINE.md`  
Anti-patterns: `layers/security-stability-baseline/v0.1/anti-patterns/ANTI_PATTERNS.md`  
Operator cards: `layers/security-stability-baseline/v0.1/operator-cards/README.md`
