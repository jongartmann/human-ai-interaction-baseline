# Rollout Checklist (v0.1)

## Day 0 — Define boundaries
- [ ] Define primary use-case(s)
- [ ] Assign human owner roles
- [ ] Set AI role level (L0/L1/L2)
- [ ] Define escalation triggers

## Day 1 — Standardize outputs
- [ ] Pick default mode behavior (Explain/Defer/Escalate/Quiet)
- [ ] Define output shapes (short, consistent)
- [ ] Add verification hooks for risky actions

## Day 2 — Prevent trust failures
- [ ] Identify over-trust risks (automation bias)
- [ ] Identify under-trust risks (rejection reflex)
- [ ] Ensure the AI can defer cleanly
- [ ] Ensure escalation is frictionless

## Ongoing — Keep it calm
- [ ] Enforce the one-screen rule
- [ ] Require resume hooks for long tasks
- [ ] Review a small sample of interactions monthly
